# FluentAR

Immersion-first language learning app with AR speaking practice.

## Languages
- English 🇺🇸
- Spanish 🇪🇸
- French 🇫🇷
- Japanese 🇯🇵
- Korean 🇰🇷
- Polish 🇵🇱

## Features
- Vocabulary flashcards with SRS
- Grammar lessons
- AR speaking challenges (hold-to-speak)
- Progress tracking
- 6 target languages

## Tech
- Expo / React Native
- Web + iOS builds

## Run
```bash
cd mobile
npx expo start
```

Web: `mobile/dist` folder

## Deploy
### Vercel
- Go to https://vercel.com/drop
- Drag `mobile/dist` folder

### Netlify  
- Go to https://app.netlify.com/drop
- Drag `mobile/dist` folder

### Cloudflare Pages
- Connect GitHub repo
- Output: `mobile/dist`
